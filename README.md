# ALX HTML CSS

This repository contains HTML and CSS projects for learning web development fundamentals.

## Project Structure

- `css_basic/` - Basic CSS styling project
  - `index.html` - Main portfolio page
  - `tweets.html` - Tweets page
  - `base.css` - Base CSS styles for layout
  - `styles.css` - Custom styles (to be developed)

## Requirements

- All files end with a new line
- No external libraries or frameworks
- W3C compliant code
- Pure HTML/CSS/JavaScript only

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
